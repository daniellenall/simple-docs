Layout

Basics

The layout of this documentation is made up of three simple divs

We have the .wrapper class, which is the base of the layout, which will contain a number of .column divs.

<div class="wrapper bg-primary"> <div class="column sidebar"> Content </div> <div class="column main bg-secondary"> Content </div> </div>

Wrapper

.wrapper class should be added to your main container div. These can contain other divs with .column classes.

Column

.column classes are flexbox elements which take up a section of the .wrapper div and should be used to create columns in your documentation page.

<div class="wrapper"> <div class="column"> Content </div> <div class="column"> Content </div> </div>

This code would create a layout with two equal columns which are sized based on their content.

Sidebar

.sidebar class should be added alongside .column, to specify the purpose of the column. Sidebar has a max width of 30%, and a minimum of 250px. to divs containing extra information. For example,

<div class="wrapper"> <div class="column sidebar bg-primary"> Sidebar content with a primary background color. The default is <span class="text-primary">green</span> </div> <div class="column"> Content </div> </div>

Main

.main class class should be added alongside .column, to specify that this is the main content of your documentation. Elements using this class will fill the remainder of the page width.

<div class="wrapper"> <div class="column"> Content </div> <div class="column main"> Put documentation here </div> </div>
